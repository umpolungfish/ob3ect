Document autopoietic bootstrap mechanics in README (2026-06-17)

Added "Autopoietic Bootstrap Mechanics" subsection inside the IMASM
Instruction Set section. The displayed 8-step bootstrap sequences are
compressed linear paths through the execution graph. The decompressed
graph shows: (1) CLINK sends an empty edge to IFIX (compositional witness
fixation); (2) CLINK sends a weighted edge back to IMSCRIB (loop
continuation); (3) IMSCRIB back-propagates to IFIX (LinFix — the
self-reference record of the completed winding). Dialetheic variant
analyzed: ENGAGR occupies the CLINK role with identical branching, but
the weighted edge carries B⬡ rather than T, which is why the dialetheic
bootstrap terminates at B⬡. Consequence stated: every winding generates
exactly two IFIX records, making the loop self-recording, not merely
self-referential. Both ASCII diagrams (compressed + decompressed) included
with legend.
